Altere o tamanho da tela para visualizar diferentes tamanhos de imagem.

Ao monitorar a rede através do Dev Tools podemos perceber que as imagens menores (e mais leves) demoram menos tempo para carregar! Isso é importante para, que caso o usuário esteja no celular, o site não carregue uma imagem desnecessariamente maior e consuma mais banda.
