Caracteres e emojis

Para escrever caracteres especiais e outros símbolos (chamados de caracteres reservados ou HTML Entities) no código HTML, utilizamos o E comercial (&) + nome do decimal ou hexadecimal.

Símbolos (& + decimal + ;):

Espaçamento (&nbsp;): html   5

Símbolo de menor/less than (&lt;): <

Símbolo de maior/greater than (&gt;)

Símbolo de libras (&pound;): £

E comercial (&amp;): &

Emojis (& + decimal ou hexadecimal + ;):

Para adicionar emojis no código utilizamos o & seguido de "#x" (indica que é um valor hexadecimal) ou apenas "#" para valores decimais, e depois o código do emoji (chamado de Codepoint). Inspecione o código para visualizar os valores hexadecimais e decimais.
